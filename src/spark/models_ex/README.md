# Ejercicios ML

Esta carpeta contiene los ejercicios que debe implementar el estudiante.

Mientras `LAB_ML` este vacio, `./lab.sh train` ejecuta estos archivos:

- `spam_detector.py`
- `engagement_predictor.py`
- `virality_classifier.py`
- `churn_predictor.py`
- `user_clustering.py`
- `follow_recommender.py`

Cuando termines de implementar un ejercicio, ejecuta:

```bash
./lab.sh train
```

Despues recarga la pestaña **Spark/ML** en la web.

Nota: no uses `./lab.sh unlock ml ...` salvo que seas profesor y quieras
mostrar las soluciones oficiales de `src/spark/models/`.
